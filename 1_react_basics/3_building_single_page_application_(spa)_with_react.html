REACT:

  • React can be used to control parts of html pages or entire page.
    • "Widget" approach on a multi-page-application. (Some) pages are still rendered on and served by a backend server.
  • React can also be usd to control the entire frontend of a web application.
    • "Single-page-application" (SPA) approach. Server only sends on HTML page, thereafter. React takes over and controls the UI.
